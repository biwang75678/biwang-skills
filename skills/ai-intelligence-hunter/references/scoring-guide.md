# 情报评分标准

## 五维评分体系

每条情报按 5 个维度评分（1-5），加权求和得到总分。

```
总分 = Impact × 0.3 + Novelty × 0.25 + Credibility × 0.2 + Timeliness × 0.15 + Relevance × 0.1
```

满分 = 5.0

---

## 维度定义与评分标准

### Impact（影响力）— 权重 0.3

对 AI 领域/产业的潜在影响程度。

| 分数 | 标准 | 示例 |
|------|------|------|
| 5 | 改变行业格局的重大突破 | GPT-4 发布、新架构超越 Transformer |
| 4 | 对主流产品/工作流有显著影响 | 重要模型开源、关键 API 变更 |
| 3 | 对特定领域/团队有实际价值 | 新工具发布、特定任务 SOTA |
| 2 | 信息有参考价值但影响有限 | 行业评论、小幅改进 |
| 1 | 仅有知识补充价值 | 教程、科普、重复报道 |

### Novelty（新颖性）— 权重 0.25

是否为首次报道或全新发现。

| 分数 | 标准 | 示例 |
|------|------|------|
| 5 | 全新发现，此前无人报道 | 独家爆料、首发论文 |
| 4 | 新角度或新组合，提供独特洞察 | 独特分析、首次关联 |
| 3 | 对已知信息的有价值补充 | 深度解读、实测报告 |
| 2 | 已知信息的重新整理 | 综述文章、总结帖 |
| 1 | 重复/转述已广泛报道的信息 | 转推、搬运 |

### Credibility（可靠性）— 权重 0.2

信息来源的可信度。

| 分数 | 标准 | 示例 |
|------|------|------|
| 5 | 官方一手来源 | OpenAI 官方博客、arXiv 论文 |
| 4 | 领域权威人士/机构 | 图灵奖得主、顶级实验室研究员 |
| 3 | 知名从业者/可靠媒体 | 知名开发者、TechCrunch |
| 2 | 有一定影响力但需验证 | 中等 KOL、社区讨论 |
| 1 | 来源不明或可信度低 | 匿名帖、未经验证的传闻 |

### Timeliness（时效性）— 权重 0.15

信息的新鲜度。

| 分数 | 标准 |
|------|------|
| 5 | 今天发布的 |
| 4 | 昨天发布的 |
| 3 | 本周内发布的 |
| 2 | 本月内发布的 |
| 1 | 超过一个月 |

### Relevance（相关性）— 权重 0.1

与用户指定的搜索主题的匹配度。

| 分数 | 标准 |
|------|------|
| 5 | 直接命中主题核心 |
| 4 | 高度相关，主题的重要组成部分 |
| 3 | 中等相关，主题的延伸方向 |
| 2 | 间接相关，需要一定推理才能关联 |
| 1 | 弱相关，仅在泛 AI 维度有关 |

## 分级分类规则

根据总分和情报性质分入以下类别：

| 类别 | 星级 | 总分参考范围 | 适用场景 |
|------|------|-------------|----------|
| **Breakthroughs** | ★★★★★ | 4.0+ | 重大突破、行业格局变化 |
| **Releases** | ★★★★ | 3.5-4.5 | 新模型/产品/工具发布 |
| **Research** | ★★★ | 3.0-4.0 | 重要研究论文、技术报告 |
| **Industry & Opinions** | ★★ | 2.5-3.5 | 行业动态、专家观点、分析文章 |
| **Trend Signals** | ★ | 2.0-3.0 | 趋势信号、社区热点、早期迹象 |

**注意**：分类不完全由总分决定。一篇论文即使总分 4.5 也应归入 Research 而非 Breakthroughs，除非它真的是改变行业的突破。分类优先看性质，总分作为排序依据。

## 评分示例

### 示例 1：OpenAI 发布 GPT-5

- Impact: 5（行业格局）
- Novelty: 5（首发）
- Credibility: 5（官方）
- Timeliness: 5（今天）
- Relevance: 5（主题核心）
- **总分**: 5.0 → **Breakthroughs** ★★★★★

### 示例 2：某 KOL 发帖分析 AI Agent 趋势

- Impact: 3（特定领域有价值）
- Novelty: 3（有价值补充）
- Credibility: 3（知名从业者）
- Timeliness: 5（今天）
- Relevance: 4（高度相关）
- **总分**: 3.0×0.3 + 3.0×0.25 + 3.0×0.2 + 5.0×0.15 + 4.0×0.1 = 0.9+0.75+0.6+0.75+0.4 = **3.4** → **Industry & Opinions** ★★
