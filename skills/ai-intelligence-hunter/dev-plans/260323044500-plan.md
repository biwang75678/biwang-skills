# AI Intelligence Hunter - 创建计划

## 需求描述

创建一个情报搜集 skill，用户给出一个主题和附加要求后，agent 执行以下流程：

1. **X/Twitter 情报搜集**（浏览器操作）
   - 使用已登录的 x.com 搜索主题相关的人、话题、帖子
   - 充分利用 X 的搜索功能和高级搜索语法
   - 利用 Grok（https://x.com/i/grok）获取情报
   - 查看话题热度、关注数等，取 top 20 相关结果
   - 对知名人士/关键 KOL 自动点关注
   - 收集当天新帖/新信息

2. **深度研究**（调用 /deep-research skill）
   - 结合 X 情报做深度搜索和发散分析

3. **产出飞书文档**
   - 关键情报来源（X 原文链接、原文简述）
   - 中文翻译
   - 话题热度/关注度数据
   - 发散性思考和深刻 insights
   - 每日新信息汇总

## 调研与决策

### 现状分析
- 无现有 skill，从零创建
- 依赖：browser skill（浏览器操作 X）、deep-research skill、feishu-create-doc skill
- X 已提前登录，可直接操作

### 方案
- SKILL.md 提供完整工作流指导
- references/ 放 X 搜索语法参考和文档模板
- 利用 browser profile="user" 操作已登录的 X
